git常用指令

  1. 全局设置

    git config --global user.name 用户名
    git config --global user.email 用户邮箱
    如果语句带参数则是设置，不带参数则为读取

  1. 初始化仓库

    git init

  2. 从远程仓库克隆仓库到本地

    git clone 仓库地址

  3. 查看当前工作目录和提交状态

    git status

  4. git 提交修改到暂存区

    提交特定文件
    git add 文件 1 文件 2
    提交所有存在改动的文件
    git add .

  5. 提交修改到本地仓库

    git commit -m 提交信息

  6. 提交修改到远程仓库

    git push

  7. 从远程仓库拉取代码

    git pull

  8. 切换分支

    git switch 分支名 或 git checkout 分支名

  9. 新建分支

    git checkout -b 分支名

  10. 合并分支

    git merge 分支名

  11. 查看当前分支版本信息

    git log
    git log --graph (以图形化（节点）展示)

  12. 删除分支

    git branch -d 分支名
    git branch -D 分支名 (强制删除)

  13. 将本地仓库与远程仓库关联

    git remote add origin https://gitee.com/LFB410/demo.git
    git push -u origin master

  14. 将分支迁移到某一版本

    git reset HEAD^ （迁移到上个版本）

    • HEAD 表示当前版本
    • HEAD^ 上一个版本
    • HEAD^^ 上上一个版本
    • HEAD^^^ 上上上一个版本
    • 以此类推…
      git reset 哈希值 （迁移到指定的版本）
